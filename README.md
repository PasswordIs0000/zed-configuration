# Zed configuration

## Symbolic links for using the configuration

Git working directory must be the current working directory for the shell.

```
ln -s $PWD/keymap.json $HOME/.config/zed/keymap.json
ln -s $PWD/settings.json $HOME/.config/zed/settings.json
```
